HANDLEIDING VOOR HET ONLINE ZETTEN VAN JE LIFESTYLE TRACKER

1. Ga naar https://vercel.com en maak een gratis account aan (of log in met GitHub).
2. Klik op 'New Project'.
3. Kies 'Import Third-Party Git Repository' (of 'Deploy Manually' als je bestanden wilt uploaden).
4. Upload de bestanden uit deze ZIP (index.html, app.js, etc).
5. Klik op 'Deploy'. Klaar!

Gebruik de 'README.txt' als geheugensteuntje.
